 * 1. Single Page application is a web app or website that interacts with the user by dynamically rewriting the current page rather than loading other pages.

 2.	Compilers are programs that convert instructions into machine-code so that they can be read and executed by a computer.

 3. A bundler is a tool that puts your code and all it's dependencies together in one file.

 4. An element is a plain object describing a component instance or DOM node.

 5.	Components are small reusable pieces of code that return an element to be rendered on a page.

 6. JSX is a syntax extension to JavaScript that is similar to a template language only it retains full power of JavaScript.

 7. Package Managers are tools that allow you to manage dependencies in your project.

 8. Content Delivery Networks deliver cached, static content from a network of servers across the globe.

 9. props are short for 'properties'. They are single values or objects containing a set of values that are passed to React Components. Also immutable.

 10. State is an object that determines how that component renders & behaves.
